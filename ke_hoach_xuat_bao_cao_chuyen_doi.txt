1. Nhận id profile từ file id.txt
2. Gọi api mở profile
3. Truy cập vào shopee.vn
4. Nếu xuất hiện //*[@href="/buyer/login?next="] thì click vào nó
5. Đợi xuất hiện //button[text()="Log In" or text()="Đăng nhập"] thì click vào nó và nghỉ 5s
6. Truy cập https://affiliate.shopee.vn/report/conversion_report
6.1 Nếu xuất hiện //*[text()="Không có dữ liệu" or text()="No Data"] thì dừng tất cả không thì tiếp tục bước 7
7. Đợi xuất hiện //a[text()="Xuất dữ liệu" or text()="Export"] thì click vào nó và nghỉ 3s
8. Truy cập https://affiliate.shopee.vn/export_management
9. Đợi (//*[@class="export-item-file-name"])[1] lấy thuộc tính href "https://affiliate.shopee.vn/api/v1/export/download?task_id=3711378" đây là link file .csv nhưng khi truy cập
sẽ tự động tải về mặc định tên file của hệ thống. Tôi muốn thay thế tên file đó bằng tên profile đang chạy. Ví dụ 
profile_name = profile1 -> file sẽ tên là "profile1.csv"
